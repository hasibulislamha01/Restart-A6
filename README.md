### 1) Difference between null and undefined

null বোঝায় no value। এই মূলত বুঝায় যে কোন ভ্যারিয়েবল এর ভ্যালু নেই। অর্থাৎ আমরা হয়ত কোন ভ্যারিয়েবল ডিক্লার করেছি কিন্তু সেই ভ্যারিয়েবল এর ভ্যালু দেইনি। এইটা সাধারণত ইন্টেনশনালি ডেভেলপার রা প্রায়ই করে থাকে যখন কোন ভ্যারিয়েবল এর ভ্যালু না দেয়াই লজিক্যালি সেফ মনে হয়। null এ ভ্যালু মিসিং নয় বরং ভ্যালু না থাকাই আসলে ভ্যালু। 

অন্যদিকে undefined হচ্ছে not defined. অর্থাৎ undefined অর্থ অই ভ্যারিয়েবল এর ভ্যালু মিসিং। এইটা সাধারণত ডেভেলপাররা ইচ্ছাকৃত ভাবে করেন না। কোনো ভ্যারিয়েবল এর ভ্যালু কোন ভাবে মিসিং থাকলে এইটা দেখা যায়।



### 2) What is the use of the map() function in JavaScript? How is it different from forEach()?

JavaScript এর map হলো একটি array method যেটা একটা array এর প্রত্যেকটা উপাদানের উপর ল্যুপ চালায়, এবং একটা নতুন array রিটার্ন করে। এটা খুবই পাওয়ারফুল একটা মেথড কেননা ল্যুপ করার মাধ্যমে আমরা অনেক কোড করা থেকে বেচে যাই। 

forEach ও এরে এর উপর ল্যুপ চালায় তবে এটা map method থেকে আলাদা এদের রিটার্ন ভ্যালু এর দিক দিয়ে। map একটা নতুন এরে রিটার্ন করলেও forEach ডিফল্ট ভাবে কিছু রিটার্ণ করেনা। forEach এরে এর প্রতিটা উপাদানের উপর ল্যুপ চালায় আর কাঙ্ক্ষিত অপারেশন চালায়। map আমরা তখন ব্যবহার করব যখন আমাদের একটা নতুন এরে দরকার হবে। forEach ব্যবহার করব যখন শুধু আমাদের এরে এর উপর ল্যুপ চালাতে হবে, নতুন কোনো এরে এর প্রয়োজন পরবে না।
