### Difference between null and undefined

null বোঝায় no value। এই মূলত বুঝায় যে কোন ভ্যারিয়েবল এর ভ্যালু নেই। অর্থাৎ আমরা হয়ত কোন ভ্যারিয়েবল ডিক্লার করেছি কিন্তু সেই ভ্যারিয়েবল এর ভ্যালু দেইনি। এইটা সাধারণত ইন্টেনশনালি ডেভেলপার রা প্রায়ই করে থাকে যখন কোন ভ্যারিয়েবল এর ভ্যালু না দেয়াই লজিক্যালি সেফ মনে হয়।
অন্যদিকে undefined হচ্ছে not defined. অর্থাৎ undefined অর্থ অই ভ্যারিয়েবল এর ভ্যালু মিসিং। এইটা সাধারণত ডেভেলপাররা ইচ্ছাকৃত ভাবে করেন না। কোনো ভ্যারিয়েবল এর ভ্যালু কোন ভাবে মিসিং থাকলে এইটা দেখা যায়।
